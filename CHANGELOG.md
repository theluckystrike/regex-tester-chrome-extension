# Changelog

All notable changes to Regex Tester Pro will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-01

### Added
- Real-time regex match highlighting as you type
- Support for all JavaScript regex flags (g, i, m, s, u, y)
- Capture group extraction (named and numbered groups)
- Find & Replace with live preview and backreference support
- Code export to 8 languages: JavaScript, Python, Java, PHP, Go, Ruby, C#, Rust
- Pattern explainer with plain-English token breakdown
- ReDoS performance safety analyzer
- 40+ curated regex examples across 8 categories
- Quick insert palette for regex building blocks
- Multi-string test mode (test pattern against multiple strings)
- Pattern history (save and recall patterns)
- Built-in regex cheat sheet
- Right-click context menu integration (test selected text)
- Dark mode with system preference detection
- Keyboard shortcuts (Ctrl+E, Ctrl+S, Ctrl+H, Esc)
- First-run onboarding experience
- User guide and help center
- Share pattern as JSON snippet
