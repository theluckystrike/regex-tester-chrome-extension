## What is Regex Tester Pro?

Regex Tester Pro is a Chrome extension that brings professional-grade regex testing to your browser toolbar. Write a pattern, see matches highlighted in real time, extract capture groups, preview find-and-replace substitutions, and export working code to eight programming languages — all without leaving Chrome and all completely offline.

Built on Manifest V3, it requires no accounts, collects no data, and makes zero network requests.

## Key Features

- **Real-time match highlighting** — results update with every keystroke
- **All regex flags** — global, case-insensitive, multiline, dotall, unicode, sticky
- **Capture group extraction** — named and numbered groups displayed clearly
- **Find & Replace** — live preview with backreference support (`$1`, `$2`)
- **Code export** — JavaScript, Python, Java, PHP, Go, Ruby, C#, Rust
- **Pattern explainer** — plain-English breakdown of every regex token
- **ReDoS safety analyzer** — detect catastrophic backtracking before it reaches production
- **40+ curated examples** — organized across 8 categories
- **Quick insert palette** — click-to-insert character classes, quantifiers, anchors, groups
- **Multi-string testing** — test one pattern against many strings at once
- **Pattern history** — save and recall patterns
- **Built-in cheat sheet** — full regex reference always one click away
- **Right-click integration** — select text on any page, right-click, test it immediately
- **Dark mode** — automatic system detection or manual toggle
- **Keyboard shortcuts** — `Ctrl+E` examples, `Ctrl+S` save, `Ctrl+H` history, `Esc` close

## Quick Start

1. Install from the [Chrome Web Store](https://chromewebstore.google.com/detail/regex-tester-pro-by-zovo/laljckjnohfcbhmlehjkcppkdfibldad)
2. Click the extension icon in your toolbar (or pin it for quick access)
3. Type a regex pattern in the top field
4. Type or paste test strings in the text area below
5. Matches are highlighted instantly — no "Run" button needed
